## 废话介绍
包含基础优化及 OptiFine 部分功能，让你玩的比较舒服的整合包。

## 环境要求
- **Java 22 或以上版本**  
  >> 如果不想安装 Java 22 或以上版本，请删除模组 **C2ME**，这样仍然需要 Java 21 或以上版本。  
- **推荐安装 Java 25**：自 **26.1-snapshot-1** 之后的游戏版本都需要 Java 25。  
- 如果你是手机用户，出现问题请自行排查。

## 安装方法
- 将整合包 (**BasicOptimized 1.21.11 1.0.0**) 直接拖入启动器窗口。  
- >> 如果无法拖入安装，可手动安装：  
  - **PCL2**: 下载 → 整合包 → 安装已有整合包  
  - **HMCL**: 下载 → 整合包 → 安装整合包 → 导入本地整合包  
  - 其它启动器大致相同。

## 模组提示
>> 如果遇到不懂、不会用的模组，可以去 [MC百科](https://www.mcmod.cn) 查找资料。  

1. **C2ME (地形生成优化)**  
   需要 Java 22 或以上版本。  

2. **ViaFabricPlus (版本转译)**  
   游戏版本过新，为保证能正常游玩部分旧服务器，安装了 ViaFabricPlus。  
   但在低性能设备上非常不友好（版本转译消耗 CPU 资源）。  
   如果游玩需要转译版本的服务器过度卡顿，请选择对应的 Minecraft 版本游玩。  

3. **DistantHorizons (遥远的地平线)**  
   优化模组，超远视距只是额外功能。  
   可将该模组视距设置为 32，原版视距拉到 12 或更低，以降低远处地形质量进行优化。  
   对核显比较友好；如果 CPU 性能过差，请删除，否则可能是负
